# INCLUDE dbt Sandbox

This repository is a space for the combined Kids First and INCLUDE teams to
iterate on design of data flows using dbt.

## Architecture Decision Records

Architecture Decision Records provide a record of key decisions and can be
found and added to in [`docs/arch`](docs/decisions/README.md).
